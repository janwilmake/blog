# True Entrepreneurship

Columbus inspires me.
True entrepreneurs are like him.
Risk-averse equals the ordinary.
To be new, one has to do things that others call ‘stupidity’.
But deep inside, you call it ‘truth’.
Entrepreneurship is an adventure.
Dare it.
Prove it.
Go over the edge and see.
